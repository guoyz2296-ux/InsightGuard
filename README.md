# InsightGuard - 愈见 PTSD 守护系统 (BUPT 雏雁计划)

### 🌟 项目简介
本项目旨在通过物联网生理数据监控与 AI 大模型情感分析，为 PTSD 患者提供无感监测与分级干预方案。

### 👥 团队分工
- **郭宇泽 (Leader)**: 后端架构、AI 接口集成、数据安全加密。
- **张逸轩**: 生理数据状态机模拟 (Mock Data)。
- **李思辰**: 前端监控大屏 (Vue + ECharts)。
- **杜如彧**: 硬件逻辑仿真、UI/UX 交互设计。
- **贾若琦**: AI Prompt 优化、分级干预文案库、结项文档。

### 🛠️ 技术栈
- Spring Boot 3 / MySQL / JPA
- Vue.js / ECharts / Axios
- AI: DeepSeek API
